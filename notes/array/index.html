An array is an object that contains a list of values.

var imgs = ["cat.jpg", "dog.jpg", "bird.jpg"];
var decades = [1980, 1990, 2000, 2010, 2020];

Array on MDN

In JavaScript, the items in an array can have any datatype, like a number, string, boolean, or even another arrary.

The elements of an array are accessed by their index number, starting with [0].

imgs[0] references "cat.jpg".

imgs[0];
imgs[1]; // "dog.jpg"
imgs[2]; // "bird.jpg"
decades[0]; // 1980
decades[3]; // 2010

Push & Pop

// push method adds one value to end of array
imgs.push("lion");
// pop removes last element
imgs.pop();
// shift removes first element
imgs.shift();

Array methods and properties

Arrays have methods and properties that help us manipulate them.

A method is a function that is associated with an object.

A property is a variable value associated with an object.

imgs.length // 3
imgs.indexOf("dog") // 1

.length is a property, because it just returns one value, the number of elements in the array.

.indexOf() is a method, because, like a function, it takes an argument and runs a block of code which returns a value.

Array with for loop

Arrays are often used with loops to repeat actions on each element in the array.
