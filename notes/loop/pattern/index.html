For the midterm we are going to put together several concepts we've been covering throughout the semester to try to make something that really justifies all the code we've been learning.

What makes p5 and programming useful? If you're a designer, you probably think it's a lot easier to draw a rectangle in Illustrator or Photoshop than p5 or JavaScript. So why are we learning to code?

What if I asked you to draw 1000 rectangles in Illustrator?

What if I then asked you to make every single rectangle a different color?

And a different size?

That's beginning to sound impossible, but in p5 it's only a few lines of code. Let's start there and then expand to see how we can make even more patterns.

Draw 1000 rectangles

Okay, our task is to draw 1000 rectangles that are all different colors and different sizes. Let's start by drawing one rectangle.

function setup() { createCanvas(640, 360); background(220); rect(0, 0, 50, 50); }

Let's add a for loop to draw 1000.

function setup() { createCanvas(640, 360); background(220); var numberOfRects = 1000; var w = width / numberOfRects; var x = 0; for (var i = 0; i < numberOfRects; i++) { rect(x, 0, w, w); x += w; } }

Change the color

Kinda hard to see huh? Maybe if we make each one a different color?

Let's grab the color randomization code from week 7:

function setup() { createCanvas(640, 360); background("white"); var numberOfRects = 1000; var w = width / numberOfRects; var x = 0; noStroke(); for (var i = 0; i < numberOfRects; i++) { var r = random(0,255); var g = random(0,255); var b = random(0,255); fill(r, g, b); rect(x, 0, w, w); x += w; } }

Make a grid

Still pretty hard to see huh? Let's draw them vertically and horizontally, then maybe we can see them more easily.

Start by using x in the for loop we already have:

function setup() { createCanvas(640, 360); background("white"); var numberOfRects = 1000; var w = width / numberOfRects; for (var x = 0; x < width; x += w) { rect(x, 0, w, w); } }

That's simpler. Now let's figure out how to draw both verical and horizontal.

First we need to calculate the rows and columns.

So solve for columns * rows = 1000

If the number of rows and columns is the same this is also columns * columns = 1000 or columns is the square route of 1000.

Fortunately, JavaScript has a built in function for getting the square root:

Math.sqrt(1000); // 31.622776601683793

Let's round up to 32, a nice even numbers, which will give us 1024 squares (a power of 2).

function setup() { createCanvas(640, 360); background("white"); var numberOfRects = 1000; var columns = Math.ceil(Math.sqrt(numberOfRects)); var rows = columns; var w = width / columns; var h = height / rows; for (var x = 0; x < width; x += w) { for (var y = 0; y < height; y += h) { rect(x, y, w, w); } } }

Simplified grid

Okay, okay, a simpler way of doing this is just deciding how many rows and columns we want to begin with. Since we've already determined we can draw > 1000 squares with 32 columns and 32 rows, let's stick with that. But we could change it later.

function setup() { createCanvas(640, 640); background("white"); var columns = 32; var rows = 32; var w = width / columns; var h = height / rows; for (var x = 0; x < width; x += w) { for (var y = 0; y < height; y += h) { rect(x, y, w, w); } } }

The important part is what happens after we decide on the number of columns.

Because columns go across the sketch and rows go up and down, we can get the right width for our rectangles by dividing total width by the number of columns. Likewise, we can get the right number of rows by dividing the total height by number of rows.

var w = width / columns;
var h = height / rows;

Then we can loop through the columns and rows testing to see if we have gotten to the bounds of the sketch.

For each column we loop through all of the rows, and then go to the next column. That's how we get our nested for loop, and our design grid.

for (var x = 0; x < width; x += w) {
	for (var y = 0; y < height; y += h) {
		// drawing functions
	}
}

Random Colors

This template should work for any grid based design you want to draw. Okay, let's bring the color back in.

function setup() { createCanvas(640, 360); background("white"); noStroke(); var columns = 32; var rows = 32; var w = width/columns; var h = height/rows; for (var x = 0; x < width; x += w) { for (var y = 0; y < height; y += h) { var r = random(0,255); var g = random(0,255); var b = random(0,255); fill(r, g, b); rect(x, y, w, w); } } }

Color palettes

Adjust the range of random colors to get different palettes. If you want mostly green, put in less red and less blue, and more green:

function setup() { createCanvas(640, 360); noStroke(); var columns = 32; var rows = 32; var w = width/columns; var h = height/rows; for (var x = 0; x < width; x += w) { for (var y = 0; y < height; y += h) { var r = random(0,100); var g = random(100,255); var b = random(0,100); fill(r, g, b); rect(x, y, w, w); } } }

Color can also be mapped to the space of the grid.

function setup() { createCanvas(640, 360); background(220); noStroke(); var columns = 32; var rows = 32; var w = width/columns; var h = height/rows; for (var x = 0; x < width; x += w) { for (var y = 0; y < height; y += h) { var r = map(y, 0, height, 0,255); var g = random(100,200); var b = random(0,20); fill(r, g, b); rect(x, y, w, w); } } }

Size variation

We can also vary the size of each shape in the gird to get a different effect.

function setup() { createCanvas(640, 360); background(220); rectMode(CENTER) noStroke(); var columns = 32; var rows = 16; var w = width/columns; var h = height/rows; for (var x = 0; x < width; x += w) { for (var y = 0; y < height; y += h) { var r = map(y, 0, height, 0,255); var g = random(100,200); var b = random(0,20); fill(r, g, b); var s = random(-10, 10); // random s rect(x, y, w + s, w + s); } } }

Choosing shapes

We can also use random and logic to change the shape function.

random(1) will be a value between 0 and 1. Half the time it will be greater than 0.5.

if (random(1) > 0.5) {
	rect(x, y, w, w);
} else {
	ellipse(x, y, w);
}
function setup() { createCanvas(640, 360); background(220); rectMode(CENTER) noStroke(); var columns = 32; var rows = 16; var w = width/columns; var h = height/rows; for (var x = 0; x < width; x += w) { for (var y = 0; y < height; y += h) { var r = map(y, 0, height, 0,255); var g = random(100,200); var b = random(0,20); fill(r, g, b); var s = random(-10, 10); // random s if (random(1) > 0.5) { ellipse(x, y, w + s); } else { rect(x, y, w + s, w + s); } } } // let's save an image save(); }
