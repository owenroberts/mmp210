In p5, we can use loop to draw many more shapes and colors with much less code. We can use loops to repeat similar tasks. First we'll look at rewriting a simple sketch using a loop.

// draw 5 circles in a line function setup() { createCanvas(640, 360); background(220); // five cicle functions with arguments ellipse(50, height/2, 40); ellipse(100, height/2, 40); ellipse(150, height/2, 40); ellipse(200, height/2, 40); ellipse(250, height/2, 40); }
// adding a variable function setup() { createCanvas(640, 360); background(220); // the first circle starts at 50 var x = 50; ellipse(x, height/2, 40); // no i can just change x instead of doing math in my head x += 50; ellipse(x, height/2, 40); // but i have to do it three more times x += 50; ellipse(x, height/2, 40); x += 50; ellipse(x, height/2, 40); x += 50; ellipse(x, height/2, 40); // this is actually more code! }
// let's add a loop function setup() { createCanvas(640, 360); background(220); // var x = 50 becomes the initial condition // x < width is as long as the x value is inside the canvas // x + 50 increment with each loop for (var x = 50; x < width; x += 50) { ellipse(x, height/2, 40); } // even more circles with 3 lines of code }

Patterns

Using a loop unlocks a lot new possiblities. The same code that we used to draw one self portrait can now be used to draw 10 self portraits, only adding a couple of lines of code.

function setup() { createCanvas(640, 360); background(220); rectMode(CENTER); for (var x = 50; x < width; x += 50) { ellipse(x, height/2, 40); ellipse(x - 10, height/2 - 10, 10); ellipse(x + 10, height/2 - 10, 10); rect(x, height/2 + 10, 30, 10, 5); } }

Interaction

Logic and interaction can be used to change the drawings at each point in the loop.

function setup() { createCanvas(640, 360); rectMode(CENTER); } function draw() { background(220); for (var x = 50; x < width; x += 50) { if (mouseX < x) { fill('plum'); } else { fill('gold'); } ellipse(x, height/2, 40); ellipse(x - 10, height/2 - 10, 10); ellipse(x + 10, height/2 - 10, 10); rect(x, height/2 + 10, 30, 10, 5); } }

Grid

The previous example shows a row of self portrait drawings divided into columns.

We can take this a step further by creating a full grid using columns and rows.

Programming make it easy to do the math to create a grid with a number of rows and columns.

function setup() { createCanvas(640, 360); background(220); // set the number of columns var columnNumber = 40; // divide the canvas by that number to get each column width var columnWidth = width / columnNumber; // draw a line at each column for (var x = 0; x <= width; x += columnWidth) { line(x, 0, x, height); } }
function setup() { createCanvas(640, 360); background(220); var columnNumber = 40; var columnWidth = width / columnNumber; // same for rows var rowNumber = 20; var rowHeight = height / rowNumber; // loop across canvas width // then loop again over height for (var x = 0; x <= width; x += columnWidth) { line(x, 0, x, height); for (var y = 0; y <= height; y += rowHeight) { line(0, y, width, y); } } }
