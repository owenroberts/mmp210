Until now, we have only been using the functions that are provided by the p5 library.

We can start writing our own functions to do more complex drawings that the defaults provided.

Defining a function in p5 looks exactly the same as it does in our previous JavaScript examples:

function setup() {
	createCanvas(640, 480);
	background("white");

	stroke("darkblue");
	drawTwoCircles(20, 20, 20, 20);
}

function drawTwoCircles(x, y, w, h) {
	ellipse(x, y, w, h);
	ellipse(x + w, y, w, h);
}

Window function

Drawing two circles instead of one isn't all that useful, but if we wanted to draw something complex like a window, it could become more useful.

Let's take one of our windows from last week's sketch and turn it into it's own function.

Drawing a window in the main setup function:

function setup() {
	createCanvas(640, 480);
	background("white");

	stroke("darkblue");

	var x = 10;
	var y = 10;
	var w = 200;
	var h = 400;
	var m = 8;

	noStroke();
	fill("pink");
	rect(x, y, w/2, h);

	fill("lightgreen");
	rect(x + w/2, y, w/2, h);

	strokeWeight(m);
	stroke("darkblue");
	noFill();
	rect(x, y, w, h);

	stroke("darkgreen");
	strokeWeight(m/2);
	rect(x + m, y + m, w/2 - 2*m, h/2);
	rect(x + m + w/2, y + m, w/2 - 2*m, h/2);
	
	rect(x + m, y + 2*m + h/2, w/2 - 2*m, h/2 - 3*m);
	rect(x + m + w/2, y + 2*m + h/2, w/2 - 2*m, h/2 - 3*m);
}

Now a separate drawWindow (can't use window because it's already a global JS object for the browser window) function:

function setup() {
	createCanvas(640, 480);
	background("white");
	
	stroke("darkblue");
	
	drawWindow(50, 10, 200, 400, 8);
}

function drawWindow(x, y, w, h, m) {
	noStroke();
	fill("pink");
	rect(x, y, w/2, h);
	
	fill("lightgreen");
	rect(x + w/2, y, w/2, h);
	
	strokeWeight(m);
	stroke("darkblue");
	noFill();
	rect(x, y, w, h);
	
	stroke("darkgreen");
	strokeWeight(m/2);
	rect(x + m, y + m, w/2 - 2*m, h/2);
	rect(x + m + w/2, y + m, w/2 - 2*m, h/2);
	
	rect(x + m, y + 2*m + h/2, w/2 - 2*m, h/2 - 3*m);
	rect(x + m + w/2, y + 2*m + h/2, w/2 - 2*m, h/2 - 3*m);
}

Loops with functions

Let's add a for loop:

function setup() {
	createCanvas(640, 480);
	background("white");
	
	stroke("darkblue");
	
	var windowColumns = 4;
	
	for (var x = 0; x < width; x += width/windowColumns) {
		drawWindow(x + 10, 10, width/windowColumns - 20, 400, 8);
	}
}

Now let's add a nested for loop:

function setup() {
	createCanvas(640, 480);
	background("white");
	
	stroke("darkblue");
	
	var windowColumns = 3;
	var windowRows = 2;
	var windowMargin = 10;
	
	for (var x = 0; x < width; x += width/windowColumns) {
		for (var y = 0; y < height; y += height/windowRows) {
		    drawWindow(x + windowMargin, y + windowMargin, width/windowColumns - windowMargin*2, height/	windowRows - 2*windowMargin, 8);
		}
	}
}
