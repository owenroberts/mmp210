Loops

A loop defines a set of commands to execute a certain number of times. It's the best way to do something repeatedly. JavaScript uses several loops statements, including do, while and for.

The syntax of a loop looks like a conditional, and in fact it uses conditions to determine how many times to run the loop.

Last week we wrote the following conditional statement and then ran it a bunch of times in the console.

var x = 0;
var y = 5;
if (x < y) {
	x++;
}
console.log(x);

This week, let's automate that process by using a while loop. Just replace the if with a while.

var x = 0;
var y = 5;
while (x < y) {
	x++;
	console.log(x);
}

The while loop is useful but it can be a little dangerous to write. This works because the loop increments x and then evaluates it, so the same value is being changed and used to determine when to stop the loop. If we wanted to do something a bunch of times other than increment x, say something like console.log("banana"); we might write a loop that never reaches it's ending condition. That will cause a stack overflow in the browser, meaning the the loop will never stop executing so our program will never finish running.

We will most often use the for loop, so that's where we'll start today. The for loop works the same way as the while loop, but it is written in a syntax that includes the beginning, update and conditional expressions in one line.

for (var i = 0; i < 5; i++) {
	console.log("This is the " + i + " loop.");
}

The for keyword indicates the beginning of the for loop. Inside the parentheses there are three statements that control the duration of the loop.

The first statement establishes the initial condition: var i = 0;.

The second statement establishes a conditional: i < 5;. As long as this condition evaluates to true, the loop will continue.

The final statement updates the condition, often with a simple increment, statement or iteration: i++;.

The code block for the loop is then contained in the curly brackets: { ... }.

for (initialization; conditional; update;) {
	loop body;
}

This is essentially like copying and pasting the same piece of code five times, while changing a number or input. Loops are an efficient way to excecute actions on a list or array of data.

In many cases the initial condition will be the value 0 as in var i = 0; ("i" in this case is shorthand for index).

But our initial condition could be anything.

for (var i = 1; i < 6; i++) {
	console.log("This is the " + i + " loop.");
}
for (var i = 99; i < 105; i++) {
	console.log("This is the " + i + " loop.");
}

Likewise, we can change the conditional and iteration statements.

for (var i = 0; i < 100; i += 10) {
	console.log(i);
}
for (var i = 10; i > 0; i--) {
	console.log(i);
}

With a for loop we don't need to worry about forgetting to increment our index value because it's written into the statement. So we can do whatever we want in the loop block.

for (var i = 0; i < 5; i++) {
	console.log("I like bananas.");
}
