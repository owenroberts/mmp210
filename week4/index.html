One of the essential components of computer based art is the possibility for interaction.

Interactive design gives the user the ability to influence the design. Sometimes the changes are small, sometimes they are large -- the artist has the ability to control how much the user can change.

In JavaScript, we use events to detect when a user interacts with a program. We can detect things like mouse movement, mouse clicks, key board inputs and change in orientation or touches in mobile devices. Today we'll start with some simple interactions.

Drawing a line

We looked at the system variables mouseX and mouseY, which track the position of the mouse, as well as pmouseX and pmouseY that tracks the previous position. We can use these simple values to create a drawing program.

function setup() { 
  createCanvas(400, 400);
	 background(220);
} 
function draw() { 
	line(mouseX, mouseY, pmouseX, pmouseY);
}

background

Notice that the background function appears in the setup function. The reason the line segments persist in the program is that we are not continuously redrawing the background, which would cover the older line segments.

We can overwrite the drawing by introducing the background with the mousePressed event.

function mousePressed() {
	background(200);
}

See how this changes if alpha is added: background(200, 127);

Key press

Let's look at another example where we can add shapes to the screen and clear them with a key press:

function setup() { 
	createCanvas(400, 400);
	background(220);
} 
function mousePressed() { 
	ellipse(mouseX, mouseY, mouseX/20 + mouseY/10);
}
function keyPressed() {
	background(200, 200);
}

Adding a drawing

We can extend the interactive drawing to apply to more shapes or combinations of shapes:

function setup() { 
  	createCanvas(400, 400);
  	rectMode(CENTER);
} 
function draw() { 
	background(200, 25);
	ellipse(mouseX, mouseY, 100);
	ellipse(mouseX - 20, mouseY - 20, 20);
	ellipse(mouseX + 20, mouseY - 20, 20);
	
	rect(mouseX, mouseY + 20, 60, 20, 10);
}

Color

We can make other aspects of the design interactive beside position:

function setup(){
	createCanvas(255, 255);
}
function draw(){
	background("black");
	fill(
		0, //red
		mouseX, //green
		mouseY //blue
	);
	ellipse(width/2, height/2, width/2);
}

We can save the color value as well:

var r, g, b;
function setup(){
	createCanvas(255, 255);
}
function draw(){
	background("black");
	fill(r, g, b);
	ellipse(width/2, height/2, width/2);
}
function mousePressed() {
	r = mouseX;
	g = mouseY;
	b = (mouseX + mouseY) / 2;
}

Size

Change the size:

var sz = 10;
function setup(){
	createCanvas(255, 255);
}
function draw(){
	background("black");
	fill("white");
	ellipse(width/2, height/2, sz);
}
function mousePressed() {
	sz += 10;
}

Another size example:

function setup(){
	createCanvas(255, 255);
}
function draw(){
	background("black");
	fill("white");
	ellipse(width/2, height/2, width - mouseX);
}
